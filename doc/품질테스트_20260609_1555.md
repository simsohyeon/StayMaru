1. 홈 진입 → 챗봇 인사+질문 노출, 진행 표시기 ①지역
2. 지역 선택→다음 / "아무 데나" / 날짜 직접 선택(달력) / 동반자 복수+무장애 / 취향 → 요약 → 코스 생성
3. **이전 버튼**으로 단계 왕복 시 메시지/선택 상태 일관
4. 다국어(ko/en/ja/zh) 라벨 정상
5. "축제 연계 + 축제 없는 지역(봉화·영양)" → 안내 토스트 노출
6. 동반자 "반려동물" → dev 콘솔에 `[tour:searchPetFriendlyPlaces] ... Forbidden` 경고가 없으면 실연동 OK. 있으면 data.go.kr "반려동물 동반여행 서비스" 활용신청 필요(미신청 시 일반검색 폴백).
