## 2. 🔴 발견 결함 및 조치

### [CRITICAL-01] 빌드 차단 — `TourApiItem` 타입에 `areacode` 누락 — **✅ 수정완료**

- **현상:** `tsc -b` 가 `src/api/tour.ts(345,27): error TS2339: Property 'areacode' does not exist on type 'TourApiItem'` 로 실패. `npm run build` 가 `tsc -b && vite build` 이므로 **운영 빌드 전체가 차단**되는 상태였음.
- **원인:** 직전 커밋(c45c689 "탐색 검색 recall 개선")에서 전국검색 결과 중 경북 항목 병합 로직에 `it.areacode` 를 사용했으나, `TourApiItem` 인터페이스에 해당 필드를 선언하지 않음. TourAPI 응답에는 `areacode` 가 실제로 존재(테스트 #1·#2에서 확인).
- **조치:** `TourApiItem` 에 `areacode?: string` 추가. 재검사 시 `tsc -b` 0 errors 확인.
- **영향:** 런타임 동작에는 영향 없었으나(JS는 필드 접근 허용) CI/배포 빌드가 막혀 있었음 → 출품 직전 치명적. 사전 발견·수정.
