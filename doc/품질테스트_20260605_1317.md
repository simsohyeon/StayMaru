- **날씨(weather.ts)** — `apis.data.go.kr/1360000/VilageFcstInfoService_2.0` 실호출 코드가 주석 처리됨(`weather.ts:50-57`). 현재는 `MONTHLY_RAIN_CHANCE` 평년값 배열(`weather.ts:15-17`)만 사용. **UI에 직접 노출되지 않고 코스 가중치(RAIN_MULT)에만 반영**되어 사용자 영향은 제한적이나, "실시간 날씨 연동"으로 오인 소지.

## 6. 결론

- **API 실연동·graceful 폴백 설계는 견고**하며, mock 데이터 제거·키 외부화 등 기본 품질은 우수하다.
- 다만 **딥링크 축제 날짜 누락(높음)**, **에러화면 문구 깨짐(높음)**, **운영화면 노출(높음)** 3건은 심사위원이 직접 마주칠 수 있는 결함이므로 **출품 전 필수 수정** 대상이다.
- 연관추천 미구독은 코드가 아닌 **활용신청 절차** 문제이며, 승인만으로 해결된다.

> 본 보고서는 코드 정독 + dev API 실호출 기준이며, 실제 브라우저 화면 렌더링(시각적 깨짐·반응형 픽셀 단위)은 별도 수동 테스트 권장.
